## How to Run

- clone project `git clone https://github.com/samiralnasser/ISDP.git`
- cd directory `cd ISDP`
- `yarn install`
- `yarn dev`
- open localhost:3000

## Requirements

- Download and install git
- Download and install node
- Download and install npm
